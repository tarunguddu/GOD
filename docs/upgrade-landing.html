Three Commands

Deploy in Shadows

1

Install Ollama

winget install Ollama.Ollama
2

Pull the Local Model

ollama pull llama3.2
3

Configure & Execute

Set the local provider in god_agent.toml:

llm_provider = "ollama"
llm_model = "llama3.2"

Then launch the self-verifying loop:

py -m god_agent.cli loop "self-check" \
  --command "py -m pytest"
